FUNCTION HELP: This game uses the same entry system as Desmos There needs to be a multiplication symbol (for example, 4(4-4+4) won't work, but 4*(4-4+4) will) All functions need parentheses (for example 4*sin 4 won't work, but 4*sin(4) will) Apart from the functions listed below, you can use +, -, /, *, ^, unary minus and parentheses special functions: sqrt - square root, automatically converts to √▔ log_x(y) - log base x of y, automatically converts to log ₓ (y) |x| - absolute value of x x% - same as x/100 x! - factorial (only for whole numbers) gcd(x,y) - greatest common divisor (2 arguments) lcm(x,y) - least common multiple (2 arguments) functions with 1 argument: usage: func(x) floor - round down ceil - round up ln - natural log log - base 10 log abs - absolute value (alias) exp - exponentiate (e^x) sin, cos, tan / tg - sine, cosine, tangent (all in radians), tangent has 2 aliases csc, sec, cot / ctg - cosecant, secant, cotangent (all in radians), cotangent has 2 aliases For all the aforementioned trigonometric functions, their: arc variants (arcsin, arccos, arctan / arctg, arccsc, arcsec, arccot / arcctg), hyperbolic variants (sinh, cosh, tanh / ctgh, csch, sech, coth / ctgh), arc hyperbolic variants (arcsinh, arccosh, arctanh / arctgh, arccsch, arcsech, arccoth / arcctgh) are available.
